# Introduction
Vanguard is a Defender remake that introduces new game mechanics that facilitates a high-risk and high-reward gameplay

## 🎮 How to Download and Play
1. Go to the **Releases** section of this repository.
2. Download the latest `.zip` file.
3. Extract the ZIP to any folder.
4. Double‑click the `.exe` file to start the game.
5. If Windows SmartScreen appears:
   - Click **More info**
   - Click **Run anyway**

#  Game Mechanics:
1.) Rescue:
- Each time a human is abducted, they can be rescued by carrying them to a drop off point located at the left side of the game space.
- Upon successful rescue, gain extra score and each three rescues gives the player an upgrade called Friend-Ship

2.) Friend-Ship
- An extra mini-ship that follows the player and provides them extra firepower
- These ships will simultaneously shoot lasers with the player

3.) Abduction
- Enemy UFOs abduct humans to the opposite side of the game space
- Each abducted humans will build up towards a swarm event
- Every 5 abducted humans will result to a swarm event

4.) Player Bomb
- A powerful resource that can destroy multiple enemies at once
- Restores bomb by moving
